# Snake Game

A simple Snake game implemented in Python using Pygame.

## Requirements

- Python 3.x
- Pygame

## Installation

1. Install dependencies: `pip install -r requirements.txt`

## How to Run

Run `python snake_game.py`

The game will prompt you in the terminal to select difficulty:
- 1: Easy (slow speed)
- 2: Medium (normal speed)
- 3: Hard (fast speed)

Press 1, 2, or 3 (no need to press Enter), then the game window will open (it should appear on top of other windows).

## Gameplay

- Use arrow keys to control the snake.
- Eat the red food to grow and increase score.
- Avoid hitting walls or yourself.

## Controls

- Arrow keys: Move snake
- Close window: Quit
