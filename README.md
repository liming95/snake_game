# Snake Game

A simple Snake game implemented in Python using Pygame.

## Requirements

- Python 3.x
- Pygame

## Installation

1. Install dependencies: `pip install -r requirements.txt`

## How to Run

Run `python snake_game.py`

Use arrow keys to control the snake. Eat the red food to grow and increase score. Avoid hitting walls or yourself.

## Controls

- Arrow keys: Move
- Close window: Quit
